# Collaboration

In Renku, a **project** is where you do your work: it holds data, code, and sessions, while a **group** is about sharing that work with others, giving members controlled access to all projects within it. You can add individual members directly to a project, or organise multiple projects under a group to manage access and visibility in one place.
